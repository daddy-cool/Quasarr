## Conventions

- Each source is documented in a file named after its two-letter uppercase identifier (e.g. `AL.md`).
- Hostnames of private or protected sources must never appear in these documents.
- When adding a new source, create a matching document here and add an entry to this index.
- Hostname and source-payload rules are defined in `AGENTS.md` (`Security And Content Rules`, `Third-Party Source Work`) and apply to every document in this folder.
